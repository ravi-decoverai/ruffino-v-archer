# Ruffino v. Archer - Case Wiki

Interactive wiki-style interface for the Ruffino v. Archer medical malpractice case (Case 3:17-cv-00725, M.D. Tennessee).

Open `wiki.html` in a browser to explore key people, timeline, and case details.

## Hosting on GitHub Pages
1. Go to repo Settings > Pages
2. Set Source to 'main' branch, root folder
3. Save - your wiki will be live at `https://<username>.github.io/<repo-name>/wiki.html`
